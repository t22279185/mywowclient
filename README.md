# 创建 Node.js 项目 过程记录

## 通过运行npm init -y，NPM 将创建一个package.json包含运行 Node.js 项目所需的最少信息。
npm init -y

####  两个依赖项。Vite 和 Three.js。

## vite
npm install --save-dev vite

##  three.js
npm install --save three

## 终端运行:
npx vite  --host
npx serve
